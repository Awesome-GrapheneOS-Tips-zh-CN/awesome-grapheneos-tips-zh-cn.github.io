安装

GrapheneOS有两种官方支持的安装方法。您可以选择推荐给大多数用户的基于WebUSB的安装页面，或选择面向更了解技术的用户的命令行安装指南。

我们强烈建议使用以上两种官方安装方法之一。第三方 安装指南往往会过时，并且常常包含受到误导的建议和 错误信息。

如果您在安装过程中遇到困难，请在GrapheneOS官方聊天频道（英文）寻求 帮助。那里几乎总有人愿意提供帮助。在请求帮助之前，请尝试 遵循指南的内容自行操作，然后针对您遇到困难的步骤 请求帮助。

命令行方法需要在拥有正确的fastboot和OpenSSH软件包的操作系统上 进行，并且您需要对整个过程有足够的了解，以避免盲目依赖我们 网站上的指示。而基于网页的安装方法只需要一个支持WebUSB的 浏览器，不需要其他任何软件，而且您仍然不必完全信任我们的 服务器基础设施，因为您可以检查启动时验证所用密钥的哈希值。
